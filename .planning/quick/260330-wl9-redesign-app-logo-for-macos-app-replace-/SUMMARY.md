# Quick Task 260330-wl9 Summary

## Goal

Replace the existing app icon with a new SVG-first logo that feels more native to a macOS developer utility and keeps the shipped asset footprint small.

## Direction

- Use a compact geometric mark instead of an illustration-heavy document icon.
- Combine code brackets with opposing swap arrows to represent switching the default editor.
- Keep fills flat or lightly graded so small sizes remain readable and PNG exports stay lightweight.

## Deliverables

- New primary source: `App/Resources/AppIcon-source.svg`
- Matching reversed and monochrome variants
- Regenerated `AppIcon.appiconset` PNG outputs
